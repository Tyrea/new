установка bootstrap3 в корневую папку сайта:
1) через командную строку, открытую из папки, набираем npm i bootstrap3
2) в папке dist вырезаем папки js,css,fonts, остальное удаляем
3) оставляем файлы с -.min.- (js и css) и шрифты
4) подключаем их в index.html
плюсы - вводим всего одну команду для скачивания; в случае возникновения неполадок где хранятся
файлы bootstrap (опасность возникает в 3 случае - при установке через CDN), сайту опасность не угрожает
минусы - у пользователя будут дополнительно подгружатся эти файлы в кэш, поэтому сайт замедлится

установка через скачивание с сайта bootstrap
1) скачиваем папку
2) оставляем файлы с -.min.- (js и css) и шрифты
3) подключаем их в index.html

плюсы - меньше лишнего скачивается (сразу только папка dist),  в случае возникновения неполадок при прямом 
подключении, сайту опасность не угрожает
минусы - нужно сделать больше действий для скачивания; у пользователя будут дополнительно подгружатся 
эти файлы в кэш, поэтому сайт замедлится

установка через CDN
1) копируем link и script для подключения
2) размещаем в нашем index.html
плюсы - быстро, сайт ускоряется за счет того, что не нужно пользователю подгружать файлы bootstrap, они скорее 
всего уже скачавались в кэш из другого сайта, содержащего эти файлы
минусы - если возникнут неполадки по адресу размещения, возникнут проблемы с сайтом.

скелет сайта копируем с сайта bootstrap, для проверки работы bootstrap  тегу h1 присваиваем класс text-center.
Если заголовок встает по центру - работает.
